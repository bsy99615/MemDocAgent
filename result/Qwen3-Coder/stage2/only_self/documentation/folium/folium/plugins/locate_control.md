## `folium.plugins.locate_control.LocateControl` · *class*

## Summary:
A control element that adds geolocation functionality to Leaflet maps, enabling users to locate their current position.

## Description:
The LocateControl class implements a Leaflet plugin control that provides geolocation capabilities to folium maps. It allows users to easily find and center their current location on the map. This control is built on top of the leaflet-locatecontrol library and integrates seamlessly with folium's rendering system through inheritance from JSCSSMixin and MacroElement.

This class serves as a distinct abstraction for map geolocation functionality, separating the concerns of UI control management from the underlying geolocation logic. It provides a clean interface for enabling automatic location detection and positioning features.

## State:
- auto_start: bool, determines whether the geolocation should start automatically when the map loads. Default is False.
- options: dict, processed keyword arguments converted to camelCase format for JavaScript compatibility
- _name: str, set to "LocateControl" to identify this element type in folium's rendering system

## Lifecycle:

- Usage: Add to a folium Figure using add_child() method; rendering automatically handles JS/CSS inclusion
- Destruction: Managed by folium's element lifecycle; no explicit cleanup required

## Example:
```python

# Create a map
m = folium.Map(location=[45.5236, -122.6750], zoom_start=13)

# Add locate control with auto-start enabled
locate_control = folium.plugins.LocateControl(auto_start=True)
m.add_child(locate_control)

# Add other map elements

# The map will automatically attempt to locate the user on load
```
